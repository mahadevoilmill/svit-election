# sviyet-election

## Vote Calculator

A simple Python script to count votes for a given number of members and candidate options.

Run:

```bash
python3 vote_calculator.py
```

## Manual Vote Casting

A dedicated manual vote casting page for casting votes by SR number and bulk updates:

- **Access**: `http://localhost:5000/manual-vote.html`
- **Features**: Single vote casting by SR number, bulk vote updates via Excel upload

Run server:

```bash
npm start
```
